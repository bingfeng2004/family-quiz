# 家庭竞答应用

一款基于浏览器的小程序，适合家庭聚会时一起玩知识竞答。

## 在线访问

直接打开：[https://bingfeng2004.github.io/family-quiz/index-cloud.html](https://bingfeng2004.github.io/family-quiz/index-cloud.html)

扫码访问：

![QR Code](family-quiz-qr.png)

## 功能

- 🏠 **首页** - 创建房间 / 加入房间 / 排行榜
- 📺 **共享大屏** - 主持人在电视/大屏上展示
- 📱 **玩家模式** - 手机扫码加入，触摸抢答
- 📊 **题库管理** - 支持多分类题目
- ☁️ **云端同步** - 通过 Supabase 实现多人实时对战

## 使用说明

### 创建房间
1. 打开首页，点击「创建房间」
2. 选择游戏模式和题目分类
3. 将房间号分享给家人

### 加入房间
1. 家人扫码或打开链接
2. 点击「加入房间」，输入房间号和昵称
3. 等待主持人开始游戏

### 主持游戏
1. 在共享大屏查看实时答题进度
2. 点击「显示答案」揭晓结果
3. 游戏结束后查看排行榜

## 技术栈

- 单文件 HTML（无框架，纯原生 JS）
- Supabase（实时数据库）
- LocalStorage（本地持久化）
- GitHub Pages（静态托管）
